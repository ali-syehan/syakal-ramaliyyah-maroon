WORKFLOW GITHUB ADA DI:
.github/workflows/build-apk.yml

CATATAN:
Folder .github diawali titik, jadi di beberapa aplikasi file manager folder ini terlihat hidden/tersembunyi.
Kalau tidak kelihatan, aktifkan Show Hidden Files.

Jika tetap tidak muncul, buat manual di GitHub:
1. Buka repo GitHub.
2. Klik Add file > Create new file.
3. Isi nama file:
   .github/workflows/build-apk.yml
4. Copy isi dari file GITHUB_WORKFLOW_build-apk.yml di root project.
5. Commit changes.

Setelah itu buka tab Actions dan jalankan workflow Build Android APK.
